# Prueba Tecnica
## Descripción
Este repositorio contiene los folders nombrados con cada uno
de los ejercicios solicitados en la prueba técnica, cada folder 
contiene un archivo de ejecución.

## Folders
test_one/ : Contiene el ejercicio 1 - Escribe un script que dado un texto tamaño N, deberá decir las veces que ese texto aparece en un
párrafo de tamaño M.

test_two/ : Contiene el ejercicio 2 - Dado un arreglo de diccionarios con diferentes atributos, se deben ordernar por prioridad encontrada en el atributo del
diccionario con nombre: “priority” ...

test_three/ : Contiene el ejercicio 3 - Crear una representación básica de una hoja de excel.
